Uso de los Floats en js


En JavaScript no existe el tipo float o double como tal.
👉 Todos los números son Number (decimales binarios).

Eso provoca errores de precisión con decimales.
Ejemplo clásico:
0.1 + 0.2
// 0.30000000000000004 😐

Ejemplo 2:
precio 20.95 y 79.9

La solución correcta: trabajar con centavos
En lugar de usar decimales, se usan enteros.
Paso 1️⃣ Convertir a centavos
20.95 → 2095
7.99 → 799


(2095+799) / 100  // se divide en 100 porque ahi 100 centavos en cada dolar 
28.94


Los cálculos con números flotantes a veces son inexactos.
Al trabajar con dinero
1. Haz el cálculo en centavos
2. Convierte de nuevo a dólares o moneda a usar



Para redeonder un decimal se usa "Math.round"
Math.round(x) 
redondea al entero más cercano, aplicando redondeo estándar si la parte decimal es > 0.5 
Ejemplo:
- 3.5 -> 4 
- 3.4 -> 3. 

sigue la logica por 3.5 esta mas puede incrementar a 4 y 3.4 no entonces se queda en 3


tambien  esta Math.floor pero no es igual
Math.floor(x) 
siempre redondea hacia abajo al entero inferior más cercano 
Ejemplo:
- 3.7 -> 3 
- -3.1 -> -4. 

este siempre escoge el inferior mas cercano no importa si el decimal est mas cerca de ser un numero grande 

Math.floor y Math.round son funciones de redondeo en programación 


tambien  esta Math.ceil pero no es igual
Math.ceil(x) 
siempre redondea hacia arriba al entero superior más cercano 
Math.ceil(2.2);
3
- 2.2 -> 3

¿Qué es una variable?

se crea una variable con estas palabras resevadas
let --> pude modificarse la variable en nombre y propiedad
const --> Solo puede modificrse sus propíedades

Reglas de sintaxis para variables

Ejemplo 1:

console.log(nuevo); //esto mostrara el 3

nuevo = 4;  //resignacion del valor de la varible
console.log(nuevo); //esto mostrara el 4

Ejemplo 2:

console.log(nuevo); //esto mostrara el 3

Ejemplo 3:
const nuevo = { nombre: "hola"};  //esto es un objeto
console.log(nuevo); //esto mostrara el { nombre: "hola"};

nuevo.nombre = "Adelso";  //resignacion de la propiedad de la varible
console.log(nuevo); //esto mostrara el { nombre: "hola"};

Atajos para sentencias if
- Operador ternario? :
- Operador de guardia &&
- Operador predeterminado ||


Operador ternario? :
 true ? "truthy" : "falsy";

 que es lo mismo que esto
 if (true) {
    "truthy";
 }else {
    "falsy";
 }

la ventaja de esto es que se puede guardar en una variable
Ejemplo:
const dato =  true ? "truthy" : "falsy";

const dato =  false ? "truthy" : "falsy";
console.log(dato); --> Dara "falsy"

valor1 && valor2
- comprueba si ambos lados son verdaderos --> true
- si un lado no es verdadero es Falso --> false

valor1 || valor2

const moneda = "EUR" || "USD";

      //Acceso a la funcion que esta dentro del objeto -- No es necesario colocarlo en un console porque es una funcion -- Es Conocido como Metodo
      producto2.funcion(); //Metodo de notacion de puntos
      producto2["funcion"](); //Metodo de Corchetes


      producto2["funcion"]();     // correcto
      producto2["funcion()"];     // incorrecto
      No es un tema de comillas… es un tema de qué es el nombre de la propiedad y qué es la ejecución de la función.

      El nombre de la propiedad es:
      "funcion"
      NO es:
      "funcion()"
      Los paréntesis no forman parte del nombre, sirven para ejecutar la función.



      Qué significa cada forma
        producto2["funcion"]();

        JavaScript lo interpreta así:
        Busca en el objeto la propiedad llamada "funcion"
        Encuentra una función
        Luego () la ejecuta


        Regla de oro (MUY IMPORTANTE)
        En objetos:
        Dentro de comillas va el nombre exacto de la propiedad
        Los paréntesis van afuera para ejecutar funciones

        Ejemplo:
        objeto["nombrePropiedad"](); 
                ▲ nombre
                                  ▲ ejecución

Más detalles sobre los objetos

1. Valor Nulo --> null
estos son muy similares a los undefined ya que representan algo que no funciona 

nulo vs indefinido
nulo = querer intencionalmente que algo esté vacío

Ejemplo:

function func(parameter ='default') {
  console.log(parameter);
}

func();          => 'default' //si esta vacio hagarra el  parametro por "default"
func(undefined); => 'default' //hagarra el parametro por "default"

func(null);       => null  //pero si el parametro es null entonces sigue siendo null

null: representa algo intencionalmente que este vacio



2. Boxeo automático (Auto-Boxing)
los obejtos pueden tener propiedades y metodos 
Ejemplo:
      const producto3 = {
          name: "camisa",  //propiedades
          precio: 799,

          metodo: function fun(){  //metodos
            console.log("metodo");
          }
        }


pero Otros valores también tienen propiedades y métodos
Ejemplo:
console.log("hola".length); // el string "hola" tambien tiene propiedades y lengt nos indica cuantas letras tiene ese string en este caso 4

pero tambien pude tener metodos 
Ejemplo:
console.log("hola".toUpperCase()); //string "hola" tambien tiene metodos, "toUpperCase()" nos da una copia de todas las letras en MAYUSCULA


todo es posible por el caracter especial de JavaScript que es "Auto-Boxing"



3. Los objetos son referencias (Objects are references)
Ejemplo:
     const objeto1 = { 
          mensaje: "hola" // esto cuando se guarda unicamente es una referencia en memoria 
      };

      const objeto2 = objeto1; // esto no hace una copia del "objeto1" si no hace una copia de la referencia del objeto que se guardo

      objeto1.mensaje = "Nuevo :)"; //Esto cambia el valor de la propiedad "mensaje" a "Nuevo :)"

      console.log(objeto1); //actualiza el objeto  
      console.log(objeto2); //El objeto2 tiene la misma referencia del objeto1 entonces eso automaticamente es lo mismo que el objeto1  

esto es conocido como copia de referencia

No podemos comparar objetos directamente
Ejemplo:
      const obejto3 = {
        mensaje: "Nuevo :)"
      }
      
      console.log(obejto3 === objeto1); //esto dara falso porque aunque tengan las mismas propiedades y valores, no son la misma referencia
                                        // y un objeto guarda una referencia 
      
      console.log(objeto2 === objeto1); // esto es true porque ambos guardan la misma referencia, No valores, referencia

-------------------------------------------LECCION 11 Arrays(Matrices) y Loops(Bucles)----------------------------------------------------------------
Matrices
  - otro tipo de valor
  - representa una lista de valores
  - en JavaScript una Matriz se represtar por [];

Ejemplo: 
      const matriz = [10, 11, 12];
      console.log(matriz);


en consola:
(3) [10, 11, 12]
  0: 10
  1: 11
  2: 12
length: 3
[[Prototype]]: Array(0)


Para obtener un valor específico en una matriz se usa => [ ] y el numero de indice que siempre empieza desde 0 y continua dependiendo de la cantidad
de elementos que contiene el array

Ejemplo:
      const matriz = [10, 11, 12];

      console.log(matriz[1]);  // en consola aparece el 11 porque es el indice 1 ya que el 10 es indice 0


con esto tambien se pude cambiar un valor dentro del array
Ejemplo:

      const matriz = [10, 11, 12];

      matriz[0] = 5; //cambio del elemento del array segun el indice
      console.log(matriz); //se actuzliza la matriz a [5, 11, 12]


Para verificar cuantos elementos existen en el array es con ".lengt"
Ejemplo:
      const matriz = [10, 11, 12];

      console.log(matriz.length); //verificacion de cuantos elementos existen en el array


Metodo ".push()" Añade un valor al final de la matriz
Ejemplo:
      const matriz = [10, 11, 12];

      matriz.push(13); //añade el elemento 13 al final del array
      console.log(matriz); //actualiza el array con el nuevo elemento añadido


Metodo ".splice()" Elimina un valor de una matriz segun su indice
    1. Índice que queremos eliminar
    2. Número de valores a eliminar

Ejemplo:
     const matriz = [10, 11, 12];

      matriz.splice(3, 1); //Elimina el elemento 13 del array porque es el indice 3 y solo le dimos 1 elemento a eliminar
      console.log(matriz); //actualiza el array con el elemento eliminado

function countWords(words) {
          const result = {};

          for (let i = 0; i < words.length; i++) {
            const word = words[i];

            // result[word] agrega o accede a una propiedad usando lo que esté
            // guardado dentro de la variable 'word'.

            // Si word = 'grape', result[word] hará result['grape']

          return result;

            function findIndex(array, word) {
              for (let i = 0; i < array.length; i++) {
                if (array[i] === word) {
                  // Instead of creating an accumulator
                  // variable and updating it in the loop,
                  // we can just return i directly, when
                  // we find it since this is a function.
                  return i;
                }
              }

              // If the function has not returned by now,
              // logically that means the word must not
              // exist in the array, so we'll return -1.
              return -1;
            }

.filter() y la funcion que se realiza en este recibe 2 parametros el valor  del array y el indice del array
[1, -3, 5]  --> [1, 5]
Números negativos eliminados
-devuelve verdadero si es positivo
-devuelve falso si es negativo

.filter()  --> numeros mayores que 0 true -- numeros menores que 0 false
    1. Crea una nueva matriz []
    2. Devuelve verdadero, => coloca el valor en la matriz
    3. Devuelve falso, => no coloca el valor en la matriz

solucion:
        console.log([1, -3, 5].filter((value, index) => {
          if(value >= 0){
            return true;
          }else if(value < 0){
            return false;
          }

        }));


solucion simplificada:
      console.log([1, -3, 5].filter((value, index) => {

        /*esto es lo mismo
          if(value >= 0){
            return true;
          }else if(value < 0){
            return false;
          }  que hacer esta linea */

        return value >= 0; //hace lo mismo que la declaracion if

      }));
  



.map() 
tenemos un array [1, 1, 3] --> se quiere hacer una copia pero que se multiplique * 2 --> [2, 2, 6]
tambien la funcion que se crea, recibe 2 parametros el valor  del array y el indice del array

.map()
  1. Crea una nueva matriz []
  2. Lo que devolvamos => se añade a la nueva matriz

.map() = transformar una matriz en otra matriz

solucion:

        }));

15m. Al calcular la fecha de entrega, omita todos los sábados y domingos.
  - Ejemplos: si hoy es viernes, 5 de octubre, y elegimos la entrega en 1 día, la fecha de entrega debería ser el lunes, 8 de octubre (omita el fin de semana).
  - Viernes, 5 de octubre + 3 días = miércoles, 10 de octubre (omita el fin de semana).
  - Lunes, 1 de octubre + 3 días = jueves, 4 de octubre (sin fin de semana intermedio).
  - Viernes, 5 de octubre + 7 días = martes, 16 de octubre (omita 2 fines de semana intermedios).


  Sugerencias: Obtenga la cantidad de días que necesita agregar. Use un bucle while y, en cada iteración, agregue 1 día y disminuya los días restantes a agregar.
  - Si la fecha es fin de semana, omita la disminución de los días restantes. (Puede usar isWeekend() del ejercicio 15e para ayudarle).

        SOLUCION:
                  //Parte para calcular la fecha
                  //Funcion que decide si es sabado o domingo
                  function isWeekend(date) {
                    const dayOfWeek = date.format('dddd');
                    return dayOfWeek === 'Saturday' || dayOfWeek === 'Sunday';
                  }


                  //Funcion para calcular la fecha
                  export function calculateDeliveryDate(deliveryOption){
                      let remainingDays = deliveryOption.deliveryDays;
                      let deliveryDate = dayjs();
                    
                      while (remainingDays > 0) {

                        // sumar 1 día
                        deliveryDate = deliveryDate.add(1, "days");
                      
                        // verificar si es fin de semana
                        if (!isWeekend(deliveryDate)) {
                          // si NO es fin de semana → restar 1 a remainingDays
                          remainingDays--;
                        }
                      }

                      
                      const dateString = deliveryDate.format("dddd, MMMM D");


                    return dateString;

                  }
